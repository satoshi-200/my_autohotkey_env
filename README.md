# My AutoHotkey Environment (Main Script)

## 概要 / Description
私のメインPCで使用している AutoHotkey v2 用のメインスクリプトです。
ホームポジションから手を動かさずに、あらゆる操作（マウス操作、記号入力、ウィンドウ管理、日付入力など）を完結させることを目的としています。

## 主なレイヤー構成 / Layer Structure
代表的な機能は以下の通りですが、これら以外にも直感的な操作を多数割り当てています。

- **変換キー (vk1C) ＋ 各種キー**
  - 【編集】削除（Delete/BS）、やり直し（Ctrl+Z/Y）
  - 【入力】記号入力、テンキー（数値入力）
- **無変換キー (vk1D) ＋ 各種キー**
  - 【マウス】カーソル移動、スクロール操作
  - 【システム】アプリ切り替え（Alt+Tab）、ウィンドウ操作（最大化・最小化等）
  - 【その他】記号入力、現在日時の自動入力
- **Spaceキー (vk20) ＋ 各種キー**
  - 【移動】文字単位/単語単位のカーソル移動、Tab/F6等によるフォーカス移動
  - 【編集】Enter、修飾キー付きEnter、F2（名前変更）、Escape
  - 【マウス】左クリック・右クリック、ページアップ/ダウン
  - 【拡張】InputHookによるさらなる階層（レイヤー）の呼び出し
- **CapsLock / カタカナひらがなキー**
  - 【拡張】InputHookを利用した、より高度な機能階層への分岐
    
## その他の仕様
- **独自ライブラリの統合**: IME制御、マルチクリップボード、VSCode連携などの外部機能を一括管理
- **入力待機（InputHook）連携**: 特定のキー（CapsLockなど）を起点とした多段的なコマンド呼び出し

## ファイル構成 / Structure
- `main.ahk`: 本ファイル（ホットキー定義の中核）
- `Lib/`: 外部関数ライブラリ群（別途参照）

## 動作環境 / Environment
- AutoHotkey v2.0+
- 日本語JIS配列キーボード推奨
